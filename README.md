# data_analysis

Analise estatistica descritiva dos arquivos `DENGBR20.csv` a `DENGBR26.csv`.

## Como executar

```powershell
node scripts\analyze_dengue.js
```

## Saidas geradas

- `artigo_analise_estatistica.md`: texto-base com metodo, resultados e observacoes para artigo.
- `outputs/tables/`: tabelas em CSV com resumo anual e distribuicoes por variaveis.
- `outputs/figures/`: graficos em SVG.
